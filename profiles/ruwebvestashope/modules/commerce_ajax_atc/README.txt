INTRODUCTION
------------

The Commerce Ajax ATC (Add to cart) module provides a flexible way
to add ajax functionality to the Drupal Commerce add to cart form.
It provides a variety of ways to render the pop-up messages
that can be configured through the UI.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/commerce_ajax_atc

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/commerce_ajax_atc

REQUIREMENTS
------------

This module requires the following modules:

 * [Commerce](https://www.drupal.org/project/commerce)

RECOMMENDED MODULES
-------------------

 * [Colorbox Load](https://www.drupal.org/project/colorbox_load):
   When enabled, it allows you to render the popup in a colorbox.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   Visit https://www.drupal.org/node/1897420 for further information.

CONFIGURATION
-------------

 * Visit /admin/commerce/config/product-types/MY_PRODUCT_TYPE/edit/display
   and edit the Variation format settings for the Add to cart form,
   and check "Enable Ajax" and click Update. Save the Display.

 * Customize the add to cart popup settings in Commerce »
   Configuration » Ajax add to cart settings.
